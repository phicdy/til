# Add icons to Asset Catalog

1. Select Images.xcassets
2. Click "+" icon
3. Select "App Icon & Launch Images" -> "New iOS App Icon"
4. Drag the icon to XCode

# Setting

Project(.xcodeproj file) -> General -> App Icons and Launch Images -> App Icons Source
